# Kubedge 

***Kubernetes, Local and Compact Learning Edge Cloud***

Kubedge local lab allows you to learn about:

- Devops, GIT, Gerrit, CICD, Docker.
- Kubernetes, MicroServices, Go, RPI, Ansible.
- LTE and 5G majors components.
- NFv/SDN Network Slicing.
